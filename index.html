laravel-crudkit

Open Source

laravel-crudkit is a CRUD framework for Laravel 5.3+. It provides a user-friendly frontend to view, create, update and delete database records in a SQL relational database. Unlike most CRUD frameworks, the frontend is constructed using a simple PHP fluent interface and can be easily retro-fit to an existing database schema.

es6-defender

Open Source

es6-defender is a clone of Classic Defender written in ES6 Javascript and created as a learning and teaching exercise in functional programming for games. The aim is for the code to be concise whilst still being readable and extensible. I've written up some of the ideas in this article.
